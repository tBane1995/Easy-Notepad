## Instalacja
1. Pobierz bibliotekę SFML i umieść ją w katalogu `C:\SFML-2.6.2`.
2. Otwórz **Wiersz poleceń (cmd)**.
3. Przejdź do katalogu projektu:
`
cd ..\..\Easy Notepad
`
4. Utwórz folder **build**:
`
mkdir build
`
5. Wejdź do folderu **build**:
`
cd build
`
6. Uruchom konfigurację projektu za pomocą **CMake**:
`
cmake ..
`
7. Zbuduj projekt:
`
cmake --build .
`

8. Plik wykonywalny znajdziesz w **build/Debug** lub **build/Release** i tam też należy przenieść pliki bibliotek **.dll** z folderu
